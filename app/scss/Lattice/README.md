Lattice
========

A dynamic CSS framework for creating modern websites.

##Core
CSS reset and clearfix.

##Grid
Responsive CSS grid.

##Forms
Basic styling for forms.

##Buttons
Basic styling for buttons.

##Typography
Basic typography styling.


Compatibility
=============

IE 9+
Chrome
Firefox
Opera

IE 8 supports unresponsive (base) version.
